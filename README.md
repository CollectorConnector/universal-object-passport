# universal-object-passport
A scalable identity layer for physical objects. Create, store, and verify digital passports with provenance, ownership history, AI‑driven valuation, and secure transfer. The foundation for universal object identity across marketplaces, collectors, and real‑world assets.
## Getting Started
1. Clone the repo
2. Run `npm install`
3. Add your Supabase keys to `.env.local`
4. Run `npm run dev`
